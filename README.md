# 🏦 Bank Management System (Python + SQL)

## 📌 Project Overview
This is a Bank Management System built using Python and SQLite.  
It allows users to create accounts, login, and perform basic banking operations like deposit, withdraw, and check balance.

---

## 🚀 Features
- 🧑 Create Account
- 🔐 Login System
- 💰 Deposit Money
- 💸 Withdraw Money
- 💳 Check Balance

---

## 🛠️ Technologies Used
- Python
- SQLite (SQL)

---

## 🧠 Concepts Used
- Functions
- Loops
- Conditional Statements
- SQL CRUD Operations (Create, Read, Update, Delete)
